Get Pyramid Params

Gets the parameters required for generating a pyramid.
Uses default values if the param is not found

Inputs

Shape Data
S Dyn Shape Data Structure
 Dyn Shape Data containing the params for this shape

Outputs

Radius
Float (double-precision)
 The radius of the shape
Height
Float (double-precision)
 The height of the pyramid
Vertices
Integer
 The number of vertices for the base of the pyramid
