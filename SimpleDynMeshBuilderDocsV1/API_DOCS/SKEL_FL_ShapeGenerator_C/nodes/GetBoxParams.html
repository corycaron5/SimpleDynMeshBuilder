Get Box Params

Gets the parameters required for generating a box.
Uses default values if the param is not found

Inputs

Shape Data
S Dyn Shape Data Structure
 Dyn Shape Data containing the params for this shape

Outputs

Dimension X
Float (double-precision)
 The X dimension for this box
Dimension Y
Float (double-precision)
 The Y dimension for this box
Dimension Z
Float (double-precision)
 The Z dimension for this box
