Save to Static Mesh

Saves the dynamic mesh into the given static mesh object, rebuilting collision for the mesh afterwards.

Inputs

In
Exec

Target
Dyn Mesh Builder Object Reference

Target Mesh
Static Mesh Object Reference
 A reference to the static mesh to be overwritten

Outputs

Out
Exec

Success
Boolean
 Whether a static mesh was successfully saved based on this dynamic mesh
